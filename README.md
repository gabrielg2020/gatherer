# gatherer

A python tool that finds all sets where Magic The Gathering cards have been printed using the Scryfall API.

## Requirements

- Python 3.6 or higher

## Usage

``` bash
python3 gatherer.py <input_file>
```

### Input format

Use the format the Moxfield gives you.
